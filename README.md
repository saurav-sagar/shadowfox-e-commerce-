E-commerce UX Enhancement

This project enhances the user experience of an e-commerce website by adding advanced product filters, sorting features, a modern UI, and a smooth multi-step checkout process.

 Features

- Category, Brand, Price, Rating, In-stock Filters  
- Sort by Price, Rating, Newest  
- Add to Cart, Update Quantity, Auto Totals  
- 3-Step Checkout: Address → Payment → Review

 Tech Used
- HTML  
- CSS  
- JavaScript
